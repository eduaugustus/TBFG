Início

O jogo

The battle for gondria é um jogo side-scroller 2D feito por 3 estudantes do SENAI para o projeto SENAI CHALLENGE, assim nasceu o The battle for gondria com o amor pela idade média e contos Medievais dos estudantes, o jogo foi feito sem usar egines como UNITY, apenas utilizando JavaScript HTML e CSS.

O projeto demorou 7 meses para ser produzido em sua totatlidade e o resultado é o que aparece no seu PC ai, esse jogo lindo com essas páginas HTML TOP TOP.

A História

The battle of gondria tem uma história de redenção, honra e disputa pela herança do trono. Com a morte de Jander, um rei seu irmão o infame Capote isola o filho herdeiro Janderson e asssume o trono, Janderson é levado a uma floresta onde este ciente de que é o verdadeiro herdeiro do trono, vivendo assim uma vida numa tribo, planejando seu retorno.

Após se preparar e conversar com seu mago ajudante Wolram, Janderson vai em busca de sua vingança
